# Microsoft Copilot 365 – Advanced Workshop (English Edition)

This deck delivers a two-hour deep dive into Microsoft Copilot 365 for advanced audiences. It focuses on AI fundamentals, prompt engineering, intelligent agents, and enterprise adoption strategies.

## How to Use the Deck

1. Install the repository dependencies from the project root:
   
   ```bash
   npm install
   ```

2. Start the development server:
   
   ```bash
   npm start
   ```

3. Open `http://localhost:8080/copilot-365-comprehensive/src/index.html` in your browser if it does not open automatically.

4. Navigate using the **Next**/**Previous** buttons, arrow keys, or by tapping on touch-enabled devices.

## Slide Highlights

- **Session Overview** – English agenda with timings and objectives.
- **AI Foundations** – Explains the Microsoft Copilot 365 architecture in English.
- **Prompt Engineering** – Translated techniques and examples.
- **AI Agents** – English walkthroughs for PDF, email, and research agents.
- **Hands-on Preparation** – Session 2 prerequisites and ideas.
